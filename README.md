# Gym Student Weight Goals 🏋️‍♂️

## 📌 Descrição do Projeto
Este projeto foi desenvolvido como uma atividade acadêmica para a disciplina de **Programação Orientada a Objetos**, com o objetivo de aplicar conceitos fundamentais de POO em um sistema simples de gerenciamento de alunos de uma academia.

O sistema permite cadastrar alunos, acompanhar seu peso atual, definir uma meta de peso e gerar relatórios básicos sobre a evolução do aluno.

---

## 🎯 Objetivos
- Aplicar conceitos de **Programação Orientada a Objetos (POO)** na prática
- Simular um sistema simples de gerenciamento de alunos de academia
- Trabalhar com entrada de dados, regras de negócio e geração de relatórios
- Organizar o código de forma modular e reutilizável

---

## 🧱 Conceitos de Programação Orientada a Objetos Utilizados
- **Classes e Objetos**
- **Encapsulamento**
- **Métodos**
- **Separação de responsabilidades**
- **Organização modular do código**

---

## 🗂️ Estrutura do Projeto
academia/
│── aluno.py # Classe responsável pelos dados do aluno
│── regras.py # Regras de negócio (metas de peso)
│── relatorio.py # Geração de relatórios
│── main.py # Arquivo principal de execução

---

## ⚙️ Funcionalidades
- Cadastro de aluno
- Registro de peso atual
- Definição de meta de peso
- Verificação se o aluno atingiu ou não sua meta
- Exibição de informações e relatórios no terminal

---
